# 🚀 GATE CSE PYQ Interface

An interactive web platform to practice **GATE CSE Previous Year Questions (PYQs)** with a clean UI, structured navigation, and mock test simulation.

---

## 🌐 Live Demo

👉 https://gatepyq-jeets.netlify.app/

---

## ✨ Features

* 📚 Subject-wise and topic-wise PYQ organization
* ⚡ Fast, responsive, and lightweight UI
* 🧠 Practice mode for concept building
* 📊 Mock test interface for exam simulation
* 🔍 Smooth navigation across questions

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
